Generador_de_graficos.py
Genera los gráficos últimos que ya consideran el solapamiento como el parámetro a estudiar, también generan los de densidad de energía para que se vea claramente porque hemos utilizado el solapamiento.
corregir.py
Genera el último Excel original hasta la fecha de esta versión de Readme, este concretamente corrige la posición incorrecta de los puntos de las líneas de geometría
Organizar_Por_Parametro.py
Este archivo busca los archivos dentro de las carpetas seleccionadas y crea una nueva a su vez crea subcarpetas para cada parámetro de estudio del proyecto, y en estas guarda todos los archivos que haya de ese parámetro en el proyecto.
renombrar_archivos.py
Renombrado de los archivos de una carpeta, se utilizo en su momento para renombrar los archivos medios que previamente se llamaban "gráficos" no se ha borrado porque es un programa muy simple y parece útil para un futuro
